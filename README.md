# GreenPulse-AI-Smart-Campus-Energy-System
AI-powered IoT energy monitoring system for sustainable campuses. Provides real-time tracking, predictive analytics, and smart alerts to reduce electricity wastage
# Smart Campus Energy Monitoring System
## AMD Slingshot Hackathon – Sustainable AI & Green Tech

---

## 📌 Problem Statement

Educational institutions face high electricity bills and energy wastage due to lack of real-time monitoring and predictive analytics. There is no intelligent system to track, analyze, and optimize energy usage.

---

## 💡 Solution Overview

Smart Campus Energy Monitoring System is an AI-powered IoT-based platform that monitors electricity consumption in real time and predicts future energy usage using Machine Learning.

The system helps institutions reduce power wastage, lower costs, and promote sustainable infrastructure.

## 🎯 Challenge Category

Sustainable AI & Green Tech

## 🚀 Key Features

- 🔹 Real-time energy monitoring
- 🔹 AI-based energy prediction
- 🔹 Abnormal usage detection
- 🔹 Smart alert system
- 🔹 Energy analytics dashboard
- 🔹 Monthly usage reports
- 🔹 Admin control panel

## 🏗️ System Architecture

IoT Energy Sensors  
        ↓  
Cloud Server  
        ↓  
Machine Learning Model  
        ↓  
Database  
        ↓  
Web Dashboard  

---

## 🔄 Process Flow

1. Sensors collect electricity consumption data  
2. Data sent to cloud server  
3. AI model analyzes usage patterns  
4. Dashboard displays analytics  
5. Alerts triggered for abnormal consumption
